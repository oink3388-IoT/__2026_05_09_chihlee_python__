# 台股四檔隨機比較工具

這是一個優化版 Dash 小工具，會隨機抽出 4 檔上市/上櫃股票或 ETF，抓取近一年 Yahoo Finance 日資料，並比較：

- 績效走勢：每檔股票起始日標準化為 100
- 核心指標：總報酬、年化報酬、年化波動、風險報酬比、最大回撤
- 相關係數：用日報酬計算股票之間的連動程度

## 安裝

```bash
pip install -r requirements.txt
```

## 啟動

```bash
python tw_stock_random_compare_app.py
```

啟動後開啟：

```text
http://127.0.0.1:8050
```

## 使用方式

- 按「重新隨機抽 4 檔」會重新抽樣。
- 搜尋股票代碼或名稱可手動加入清單。
- 清單最多保留 4 檔；新增第 5 檔時會自動移除最早的一檔。
- 點股票標籤右側的 `x` 可移除。

## 注意

資料來源為 Yahoo Finance。若部分台股代碼暫時抓不到資料，工具會顯示提示並允許重新抽樣。此工具用於量化觀察，不構成投資建議。
